# portfolio-api

## Development

Install dependencies:

```bash
pnpm install
```

Start local development:

```bash
pnpm dev
```

Deploy:

```bash
pnpm deploy
```

## API Endpoints

`GET /api/projects`

`GET /api/project/:id`

`GET /api/services`

`GET /api/service/:id`

`GET /api/skills`

`GET /api/skill/:id`
